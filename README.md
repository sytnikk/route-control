# route-control


## Running the app
```bash
$ git clone --recurse-submodules https://github.com/sytnikk/route-control.git
$ cd route-control
$ docker-compose -f "docker-compose.yml" up -d --build
```
App link: http://localhost:80

## Shutdown the app

```bash
$ cd route-control
$ docker-compose -f "docker-compose.yml" down
```
